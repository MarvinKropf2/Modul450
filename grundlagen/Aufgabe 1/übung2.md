# Aufgabe 1 – Übung 2  
Fehler – Mangel – hoher Schaden

## Beispiel Softwarefehler
Ein Fehler ist eine Abweichung vom Soll Verhalten.  
Beispiel: Rabatt soll ab 100 Franken gelten, der Code gibt aber bereits ab 50 Franken Rabatt.

## Beispiel Softwaremangel
Ein Mangel ist eine nicht erfüllte berechtigte Erwartung.  
Beispiel: Preis wird korrekt berechnet, aber auf dem PDF ohne Währung und ohne Format angezeigt.

## Beispiel hoher Schaden
Beispiele für sehr grossen Schaden durch einen Softwarefehler:

• Falsche Medikamentendosis in einer Spital Software  
• Fehler in einer Brems oder Flugsteuerung  
• Algorithmus im Börsenhandel löst falsche Orders aus
