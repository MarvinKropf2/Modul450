# Aufgabe 1 – Übung 1  
Formen von Tests in der Informatik

| Testart          | Erklärung                                                         | Durchführung                                               | Beispiel                                         |
|------------------|-------------------------------------------------------------------|-------------------------------------------------------------|--------------------------------------------------|
| Unit Test        | Test einer einzelnen Funktion oder Klasse                          | Automatisiert im Code                                       | Java JUnit Test für eine Methode                 |
| Integrationstest | Test wie mehrere Komponenten zusammenarbeiten                      | Module zusammen starten und Abläufe testen                  | API ruft Service und Datenbank                   |
| Systemtest       | Test des gesamten Systems aus Sicht der Benutzer                   | Ganze App in Testumgebung durchklicken                     | Checkout Flow im Webshop                         |
| Akzeptanztest    | Prüfen ob das System für den Kunden korrekt funktioniert           | Kunde testet echte Businessfälle                           | Product Owner testet User Stories                |
| Regressionstest  | Sicherstellen dass bestehende Funktionen nach Änderungen weitergehen | Vorhandene Tests erneut ausführen                           | Alle Tests nach jedem Release                    |
